# testetecnico
Esse é um projeto  utilizando  React com layout fornecido pela  Fasters Soluções.

# instalação

Após descompactar  o arquivo  .zip, deve  abrir  o terminal nessa pasta e entrar no projeto  utilizando  cd nome da pasta,  após isso é necessário instalar as  dependências com o  comando npm install.

#Execução  do  projeto

para  execução  utilize  o  comando npm run  start.

#Tecnologias

* Javascript
* React
